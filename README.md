# Sistema Help Desk

Sistema de gerenciamento de chamados desenvolvido durante o curso técnico de TI.

## Tecnologias

- PHP
- HTML5
- CSS3
- MySQL

## Funcionalidades

- Login de usuários
- Cadastro de usuários
- Abertura de chamados
- Edição de chamados
- Exclusão de chamados
- Controle de acesso

##Estrutura do Banco de Dados

O script do banco de dados está disponível no arquivo:

`helpdesk_simples.sql`

## Como executar

1. Clone o repositório
2. Inicie o Apache e o MySQL no XAMPP.
3. Importe o arquivo `helpdesk_simples.sql` no phpMyAdmin.
4. Configure a conexão com o banco de dados em `conexao.php`.
5. Acesse o sistema pelo navegador.
